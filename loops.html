FizzBuzz Challenge
Write a program that prints all numbers from 1 to 100. For all numbers divisible by 3 print Fizz, for all numbers divisible by 5 print Buzz. For any numbers divisible by both 3 and 5 print FizzBuzz, otherwise just print the number.
Chessboard
Part 1
Write a program that creates a string that represents an 8X8 grid, using new-line characters to separate lines. At each position of the grid there is either a space or a # character forming a chessboard.
Part 2
Define a binding size = 8 and update the program to work for any size outputting a grid of the given width and height.
